# 💉🇶🇪 مستشار التحصين الصحي الموسع - اليمن

## Yemen EPI Immunization Advisor Bot

[![Build Flutter APK](https://github.com/mohammedshoqi123-art/EPI-Bot/actions/workflows/build.yml/badge.svg)](https://github.com/mohammedshoqi123-art/EPI-Bot/actions/workflows/build.yml)

تطبيق Flutter ذكي للاستشارة حول برنامج التحصين الصحي الموسع في اليمن 🇾🇪

## ✨ المميزات

### 🤖 روبوت استشاري ذكي (7/10 ذكاء)
- **144+ موضوع معرفي** شامل
- **محرك NLP عربي** متقدم (تطبيع، مرادفات، كشف نفي)
- **مدير سياق عميق** — يحفظ تفاصيل الطفل
- **أسئلة توضيحية** تلقائية
- **فهم النفي** ("ما أبي أطعمه")
- **كشف شدة الأعراض** (خفيف/متوسط/شديد)
- **15+ نية** مكتشفة
- **ذاكرة 20 دورة** محادثة
- **يعمل 100% بدون إنترنت** 📵

### 📅 جدول التطعيمات الكامل
- 20+ تطعيم بتفاصيل كاملة
- 11 مرض مغطى بالتفصيل
- تصفح حسب مرحلة العمر
- تفاصيل كل تطعيم (المرض، الجرعة، مكان الحقن)

### 📋 بطاقة التطعيم الرقمية
- تسجيل بيانات الطفل
- متابعة نسبة الإنجاز
- شريط تقدم دائري
- 22 محافظة يمنية

### 📚 قاعدة المعرفة الشاملة
- أنواع المناعة (فطرية/تكيفية)
- سلسلة التبريد و VVM
- حالات خاصة (مبتسرين، سكري، قلب، HIV)
- أساطير مُدحضة (التوحد، العقم)
- تغذية ورضاعة
- تاريخ التحصين في اليمن
- تطعيمات السفر والطوارئ

## 🚀 التشغيل

### المتطلبات
- Flutter SDK 3.0+
- Dart 3.0+
- Android Studio أو VS Code

### الخطوات

```bash
# 1. استنساخ المشروع
git clone https://github.com/mohammedshoqi123-art/EPI-Bot.git
cd EPI-Bot

# 2. تشغيل سكريبت الإعداد
bash setup.sh

# 3. تشغيل على جهاز
flutter run

# 4. بناء APK
flutter build apk --release
```
